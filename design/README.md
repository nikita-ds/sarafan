# Design assets

Эта папка — источник референсов для вёрстки прототипа.

## Структура
- `Иконки/`, `Логотип/`, `Фоны/` — исходники из дизайн-пака.
- `screens/` — рендеры экранов из `App Sarafan.pdf` (page-01..page-09).

## Как использовать
- Открывайте `design/screens/page-*.png` рядом с `index.html` и подгоняйте отступы/размеры/шрифты.
- Экранные тексты и порядок экранов соответствуют PDF.
