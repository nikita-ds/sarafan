# SARAFAN — кликабельный прототип (mobile web)

## Открыть локально
- Быстро: откройте `index.html` в браузере.
- Надёжнее (через локальный сервер):
	- `cd /Users/nikita/Downloads/sarafan`
	- `python3 -m http.server 5173` (если порт занят — используйте, например, `5174`)
	- открыть `http://localhost:5173`

## Открыть с телефона
- Запустите сервер как выше.
- Телефон и Mac должны быть в одной Wi‑Fi сети.
- Узнайте IP вашего Mac (например, `192.168.0.10`) и откройте на телефоне:
	- `http://<IP>:5173`

## Флоу прототипа
- 2 роли: Клиент и Бизнесмен.
- Регистрация: выбор роли → ФИО/телефон/email → город → фото.
- Клиент: поиск/категории → список → карточка бизнеса → создать запрос.
- Бизнесмен: лента запросов по городу → «Откликнуться».
